> INSTALLATION PROCEDURES

SYSTEM REQUIREMENTS:

  • MINECRAFT JAVA EDITION 1.12.2
  • FORGE VERSION 14.23.5.2860
  • MINIMUM 8GB RAM ALLOCATED (12GB RECOMMENDED)

INSTALLATION STEPS:

  1. DOWNLOAD MODPACK FROM CURSEFORGE
  2. INSTALL CURSEFORGE LAUNCHER OR MULTIMC
  3. IMPORT MODPACK ZIP FILE
  4. ALLOCATE APPROPRIATE RAM IN LAUNCHER SETTINGS
  5. LAUNCH AND INITIALIZE WORLD GENERATION

OPTIONAL JVM ARGUMENTS FOR OPTIMIZATION:

-XX:+UseG1GC -Dsun.rmi.dgc.server.gcInterval=2147483646 -XX:+UnlockExperimentalVMOptions -XX:G1NewSizePercent=20 -XX:G1ReservePercent=20 -XX:MaxGCPauseMillis=51 -XX:G1HeapRegionSize=32M
